Progressive Enhancement Architecture

Start with simple chat and gradually scale up to enterprise-level autonomous agents. Our 4-level approach ensures you only implement the complexity you need.

Level 1

Conversational Agent

Simple chat, Q&A, onboarding, FAQ bots. Perfect for getting started.

$response = Agent::simpleChat('Hello world');
Level 2

Agent with Tools

Functions, APIs, calculations, file operations. Plus OpenAI official tools.

$agent->registerCodeInterpreter();
$agent->registerRetrieval(['k' => 3]);
$agent->registerWebSearch();
Level 3

Multi-Agents

Multiple specialized agents collaborate seamlessly with handoffs and workflows.

$runner->setHandoffOrchestrator(
    app(HandoffOrchestrator::class)
);
Level 4

Autonomous Agents

Self-monitoring, decision-making agents that can initiate actions and adapt.

$agent = Agent::create([
    'mode' => 'autonomous',
    'autonomy_level' => 'high'
]);

Quick Start

Get up and running in minutes with our progressive enhancement approach

Installation

1. Install via Composer

composer require sapiensly/openai-agents

2. Publish Configuration

php artisan vendor:publish --tag=config --provider="Sapiensly\\OpenaiAgents\\AgentServiceProvider"

3. Set Environment Variables

OPENAI_API_KEY=your-openai-api-key-here
OPENAI_MODEL=gpt-4o
AGENTS_PROGRESSIVE_LEVEL=1

Usage Examples

Simple Chat

use Sapiensly\OpenaiAgents\Agent;

$response = Agent::simpleChat('Hello world');
echo $response;

With Tools

$runner = Agent::runner();
$runner->registerTool('calculator', function($args) {
    return eval("return {$args['expression']};");
});
$response = $runner->run('Calculate 15 * 23');

OpenAI Official Tools

$agent = Agent::create(['model' => 'gpt-4o']);
$agent->registerCodeInterpreter();
$agent->registerRetrieval(['k' => 3]);
$response = $agent->chat('Analyze this data');
